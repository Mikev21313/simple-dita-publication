Welcome to My Simple DITA Publication

This is a simple one-page DITA publication deployed to GitHub Pages.

DITA (Darwin Information Typing Architecture) is an XML-based format for authoring and publishing technical content.

Key Features

  • Structured Content: DITA helps organize content in a structured way.
  • Reusability: Content can be reused across multiple documents.
  • Separation of Content and Presentation: Content is separate from its formatting.

About This Example

This example demonstrates how to publish DITA content to GitHub Pages.
